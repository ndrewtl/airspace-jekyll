DFA Admissions Prep

Pre Interview Project

In order to make sure all students can maximize their benefit from the course, all students must be at an expected level of technical competency before joining one of DFA's cohorts.

Our Pre-Interview Project tests that you are familiar with front end development enough to build at least a small app using an API to interact with data. In order to work on this project you'll need to be familiar with these technologies:

  • ReactJS (no Redux necessary, just basic React is fine)
  • JavaScript ES6
  • HTTP Requests
  • HTML/CSS
  • How to use the command-line/terminal enough to navigate around and execute programs

Resources to get familiar with the above

Although this course isn't to teach you the basics above the above, it's possible you need to ramp up on any/all of the above in order to pass the interview for this course. Do not at all feel like you need to go through all of these resources. After you get the Pre-Interview Project you may just need to pick specific pieces to ramp up on in order to work on it. Also, if you come across anything useful on your own and would like to contribute to this list, shoot it over.

ReactJS
  • Official Facebook guide and tutorial
  • ReactJS Crash Course
JavaScript/ES6
  • Top 10 ES6 Features Every Busy JavaScript Developer Must Know
  • Coderbyte - Isolated Problem Solving with JS (if you don't feel like you know JavaScript like the back of your hand, going through the motions on these can help ease you into it)
  • "You Don't Know JS" - Book Series of various fundamental concepts of JavaScript
HTTP Requests
  • HTTP: The Protocol Every Web Developer Must Know
